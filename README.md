# نظام محاسبة المتاجر التجارية

مشروع متكامل لنظام محاسبة إلكتروني للمتاجر التجارية.

## المميزات

- إدارة المنتجات والمخزون
- تسجيل المبيعات والمشتريات
- تقارير مالية متقدمة
- واجهة مستخدم عربية متجاوبة

## التقنيات المستخدمة

### Backend
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy

### Frontend
- React.js
- CSS3
- Axios

## طريقة التشغيل

### المتطلبات الأساسية
- Docker
- Docker Compose
